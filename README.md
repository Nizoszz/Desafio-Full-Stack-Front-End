## Instalação

```bash
$ yarn install
```

## Inicialização do app

```bash
$ yarn dev
```

## Após a execução de inicialização renderizara no terminal:

```bash
VITE v4.3.8

  ➜  Local:   http://localhost:<port>/
  ➜  Network: use --host to expose
  ➜  press h to show help

```
